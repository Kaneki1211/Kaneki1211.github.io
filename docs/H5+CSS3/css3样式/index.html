四、CSS3--渐变、过渡、变换

1、渐变​

线性渐变​

background-image：linear-gradient（方向，颜色 1，颜色 2）
方向（水平垂直） to top, to bottom, to left, to right
单词（对角方向） to left top, to 水平方向+垂直方向
角度值（deg） 正值 ：顺时针 负值 ： 逆时针

.box1 {
  background-image: linear-gradient(180deg, pink, #bfa);
}
.box1 {
  background-image: linear-gradient(rgb(240, 15, 53) 30%, blue 70%);
}
/* 
        红色实体 30%
        渐变色 70%-30%=40%
        蓝色实体 100%-70%=30%
*/
.box2 {
  background-image: linear-gradient(rgb(240, 15, 53) 40%, blue 30%);
}
/* 
        红色实体  40%
        渐变色 0%
        蓝色实体 60%
*/

径向渐变​

background-image:radial-gradient(渐变形状 大小 at 渐变中心点，start-color，...，last color)

渐变形状: 正圆 circle 椭圆 ellipse

大小:px % closest-side 最近边 farthest-side 最远边 closest-corner 最近角 farthest-corner 最远角

如果设置大小是百分比：设置两个值 水平 垂直
如果形状是正圆：大小 水平和垂直要相等

.box1 {
  background-image: radial-gradient(circle, red, yellow, blue);
}
.box2 {
  background-image: radial-gradient(circle at top right, red, yellow, blue);
}

重复渐变​

重复线性渐变
background-image：repeating-linear-gradient（方向，颜色 0，颜色 10%，颜色 20%）
重复径向渐变
background-image：repeating-radial-gradient(渐变形状 大小 at 渐变中心点，start-color，...，last color)

div {
  width: 200px;
  height: 200px;
  margin: 0 auto;
  background-image: repeating-linear-gradient(red 0, yellow 10%, green 20%);
}

2、过渡​

transition：1.参与过渡的属性 2.执行时间 3.延迟时间 4.速度动画

触发条件
1.需要用户动作事件触发
2.属性值需要发生改变，初始值，最终值

  • 参与过渡的属性
    transition-property：all
  • 执行时间（s/ms）
    transition-duration
  • 延迟执行
    transition-delay
  • 速度动画
    • ease 默认速度，逐渐变慢
    • linear 匀速
    • ease-in 加速
    • ease-out 减速
    • ease-in-out 先加速后减速
    • steps()自定义设置几步完成
    • cubic-bezier(0,0.91,1,0.43)设置贝塞尔曲线
div {
  width: 100px;
  height: 100px;
  background: red;
  /* transition: width 1s; */
  transition-property: width, height;
  transition-duration: 0.5s, 0.5s;
  transition-delay: 0s, 0.5s;
}
div:hover {
  width: 200px;
  height: 200px;
  background: green;
}

3、2D 的变换​

2D 的位移​

位移：相对定位（占位，原位置）

  • transform:translate() 同时设置水平和垂直方向的位移

  • transform:translateX(x) 设置水平方向的位移

  • transform:translateY(y) 设置垂直方向的位移

  • 简写 transform: translate(200px) x 轴位移 200px

p {
  width: 100px;
  height: 100px;
  background: #bfa;
  transition: 1s;
}
div:hover p {
  transform: translate(200px, 200px);
}

2D 的缩放​

缩放（单位是倍数）

  • scaleX(x) 设置水平方向的缩放比例
  • scaleY(y) 设置垂直方向的缩放比例
  • scale() 同时设置水平方向和垂直方向的缩放比例
  • 支持负值，反方向缩放（倒过来）
  • 简写： transform: scale(2) 设置 x 轴和 y 轴
p {
  width: 100px;
  height: 100px;
  background: #bfa;
  transition: 1s;
}
div:hover p {
  transform: scale(2, 2);
}

2D 的旋转​

2d 属性的旋转功能（deg 角度）

  • rotate()顺时针和逆时针
  • rotateX()上下旋转
  • rotateY()左右旋转
p {
  width: 100px;
  height: 100px;
  background: #bfa;
  transition: 1s;
}
div:hover p {
  transform: rotate(360deg);
}

2D 的倾斜​

2D 的倾斜（单位是 deg 角度）

  • skewX() 元素在水平方向倾斜
  • skewY() 元素在垂直方向倾斜
  • skew() 元素在水平方向和垂直方向的倾斜
p {
  width: 100px;
  height: 100px;
  background: #bfa;
  transition: 1s;
}
div:hover p {
  transform: skew(0, 30deg);
}

4、3D 的变换​

开启 3D 空间​

transform-style: preserve-3d

3D 的景深​

perspective 景深，透视（加强 3d 的视觉效果，作用近大远小）

设置：

  • 父元素：每一个子元素的景深效果各不相同 perspective：1000px
  • 子元素：每个子元素的景深效果都一样 transform：perspective

3D 的位移​

3d 的位移：z 轴，正前负后

  • translateZ（z）
  • transform3d（x,y,z）第 1 个参数对应 X 轴，第 2 个参数对应 Y 轴，第 3 个参数对应 Z 轴，参数不允许省略
  • 注意：设置 3d 属性父元素需要开启 3d 空间 transform-style：preserve-3d
div {
  width: 400px;
  height: 300px;
  background: rgba(0, 0, 0, 0.4);
  border: 2px solid #000;
  margin: 100px auto;
  transform-style: preserve-3d;
  /* 开启3d空间 */
}
section {
  width: 200px;
  height: 100px;
  border: solid green;
}
div:hover section {
  transform: translateZ(-300px);
}

3D 的旋转​

  • rotateX() 指定对象在 x 轴上的旋转角度
  • rotateY() 指定对象在 y 轴上的旋转角度
  • rotateZ() 指定对象在 z 轴上的旋转角度
  • rotate3d（x，y，z，角度值） x,y,z 表示最后一个参数的倍数
div {
  width: 400px;
  height: 300px;
  border: 2px solid #000;
  margin: 100px auto;
  perspective: 1000px;
  transform-style: preserve-3d;
  /* 开启3d空间 */
}
section {
  width: 200px;
  height: 100px;
  background-color: green;
  transition: 2s;
}
div:hover section {
  transform: rotateZ(360deg);
}

3D 的缩放​

3d 的缩放：z 轴就是盒子厚度缩放，结合旋转一起

  • scaleZ() 指定对象的 z 轴缩放
  • scale3d() 指定对象的 3D 缩放。第 1 个参数对应 X 轴，第 2 个参数对应 Y 轴，第 3 个参数对应 Z 轴，参数不允许省略
div {
  width: 400px;
  height: 300px;
  background: rgba(0, 0, 0, 0.4);
  border: 2px solid #000;
  margin: 100px auto;
  transform-style: preserve-3d;
  transform: rotateX(15deg) rotateY(15deg);
  /* 开启3d空间 */
}
section {
  width: 200px;
  height: 100px;
  border: solid green;
}
div:hover section {
  transform: scaleZ(2);
}
