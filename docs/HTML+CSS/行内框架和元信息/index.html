四、行内框架和元信息

1、行内框架 iframe​

<iframe src="" frameborder="0"></iframe>
<!-- 引入广告地址-->

<a href="" target="iframe"></a>
<!-- iframe指定a标签的打开位置-->

<iframe src="" frameborder="0" name="iframe"></iframe>
<!-- iframe 指定提交表单的位置-->

2、html 的全局属性​

  • id 给标签取名字，区分标签内容，唯一性
  • class 给标签起名字，名字可以重复设置
  • title 悬浮提示框 （超链接）
  • style 设置 css 样式
  • dir 控制文本对齐的方向
  • lang 触发浏览器的翻译功能

3.mete 元信息​

mete 的属性

  • name
    • keywords 关键字
    • description 网页的描述信息
  • content 描述的内容 网页优化三剑客
<!-- 网页标题 -->
<title></title>
<!-- 网页的描述 -->
<meta name="discription" content="" />
<!-- 网页搜索的关键字 -->
<meta name="keywords" content="" />
