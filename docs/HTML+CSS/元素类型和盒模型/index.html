八、元素类型和盒模型

1、元素的类型​

  • 块元素
    • 上下显示，独占一行，可以自动换行
    • 可以识别宽高，可以识别 4 个方向的 margin 和 padding
    • 块元素当做其他元素的容器
    • 特殊：p，h1~h6,dt 不要嵌套其他的块元素
  • 行内元素(span,em,strong,sub,sup,ins,del,a)
    • 水平显示，不会自动换行
    • 宽高由内容撑开，无法设置 width 和 height，可以识别水平方向的 margin 和 padding
  • 行内块元素(img,intput,textarea,select)
    • 水平显示，不会自动换行
    • 都有自己本身的宽高，可以识别设置的 width 和 height，也可以识别 4 个方向的 margin 和 padding

2、元素类型的转换​

元素类型可以相互转换

display​

  • block 块元素
  • inline 行内元素
  • inline-block 行内块元素
  • none 隐藏元素 不占位隐藏

注意：行内块元素不能转换为行内元素

3、水平居中​

  • 文本居中

    text-algin （文本，行内元素，行内块元素）

  • 盒子居中

    margin：0 auto （块元素）

4、垂直对齐​

垂直对齐 vertical-algin （行内或行内块）​

  • top 顶部对齐
  • bottom 底部对齐
  • middle 垂直居中
  • baseline 基线对齐
div {
  border: #000 solid 1px;
  height: 800px;
  width: 600px;
  text-align: center;
  margin: 0 auto;
  /* position: relative; */
}
img {
  /* position: absolute; */
  height: 100px;
  width: 200px;
  /* top: 350px; */
  /* left: 200px; */
  vertical-align: middle;
}
span {
  display: inline-block;
  height: 800px;
  vertical-align: middle;
}
<div>
  <img src="./images/bg1.png" alt="" />
  <span></span>
</div>
   总结：vertical-align
       1.行内元素或者块元素
       2.相互参照居中，需要有参照物
       3.vertical-algin：middle

   取消图片下方自带的间隙（原理，图片的对齐默认是基线对齐）
       vertical-algin：top，bottom，middle

5、盒模型​

盒模型的组成​

盒子结构分为 4 部分​

  • 内容（width 和 height）
  • 内边距（padding）设置内容与盒子的距离
  • 边框线（border）设置盒子的边框
  • 外边距（margin）设置盒子与盒子之间的距离

内边距 padding​

  • 作用：设置盒子与内容之间的距离

  • 注意： -内边距会撑大盒子，如果设置 padding 想要总宽高不变 width 和 height 减去加上 padding 的大小 -内边距不能设置负值

  • 设置 padding

      单个方向
          padding-left
          padding-right
          padding-top
          padding-bottom
      多个方向
          padding：
          1个值，4个方向
          2个值，上下  左右
          3个值，上  左右  下
          4个值，顺时针  上  右  下  左

外边距 margin​

  • 作用：设置盒子与盒子之间的位置关系

  • margin: 0 auto

    • 让盒子水平居中
    • 注意：一般情况下 auto 只能设置水平方向的外边距自动居中，上下方向 auto 一般情况不管用，（弹性布局，定位，可以设置上下 margin 也 auto）

  • 自己移动

    • margin-left
      • 正 右
      • 负 左
    • margin-top
      • 正 下
      • 负 上

  • 设置其他盒子距离

    • margin-bottom
      • 正：让其他合适往下移动
      • 负：让其他盒子往上移动
    • margin-right
      • 正：让其他盒子往右移动
      • 负：让其他盒子往左移动

  • 外边距常见 bug

    1.两个盒子上下的外边距会重叠，谁的值大设谁
    解决：给一个盒子设置边距

    2.外边距的margin-top的 传递问题
    当父元素下的第一个子元素设置margin-top的时候，margin-top会错误传递给父元素
    解决方法：
        父元素：
           1. 阻隔作用
               border
               padding
           2. 触发BFC（是一个独立的渲染区域，有自己的布局规则）
                overflow：hidden
                display：inline-block
           3. 浮动 display：float
           4. 定位 position
           5. 弹性布局 display：flex

边框 border​

border：粗细 样式 颜色

  • 单一方向
    • border-left
    • border-right
    • border-top
    • border-bottom
  • 取消边框
    • border：none/0
  • 粗细
    • border-width
  • 样式
    • border-style
      • none 默认值
      • solid 实线
      • dotted 点线
      • dashed 虚线
      • double 双线
  • 颜色
    • border-color

6、Max，Min 高度和宽度的设置​

  • 最大宽度 max-width：700px（小于等于 700）

  • 最小宽度 min-width：700px（大于等于 700）

  • 高度设置 height：auto（高度由内容撑开） height：100%（父元素高度的百分比）

  • 最小高度（在高度 auto 上的一个优化） min-height

  • 最大高度 max-height

  • 使用场景：

  /* 盒子的高度等于浏览器窗口的高度 */
  html，body{
  height：100%
  }
